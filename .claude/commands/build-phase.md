# Build Phase

Execute a specific build phase from the Brand Me Now v2 BUILD-GUIDE.

Read `docs/prd/BUILD-GUIDE.md` and execute the build step specified by the user. Follow the exact prompt, create all files listed, and run the verification commands.

Before starting:
1. Read the BUILD-GUIDE.md to find the specified step
2. Read the referenced PRD doc(s) completely
3. Execute the build prompt exactly as specified
4. Run all verification commands
5. Report results

Usage: /build-phase <step-number>
Example: /build-phase 1.1
